# HK20_EduCore_TechLytics
🎓 EduCore

AI-Based Assignment Analytics & Intelligent Feedback System

🚀 An intelligent web application designed to automate assignment evaluation and provide AI-powered feedback for teachers.

⸻

📌 Overview

EduCore is an AI-powered academic evaluation platform that helps teachers analyze student assignments efficiently.

The system processes uploaded assignment data (CSV format), evaluates performance using AI-based logic, and generates:

	•	📊 Scores
	•	📝 Personalized Feedback
	•	📈 Performance Insights
	•	💡 Improvement Suggestions

It reduces manual workload and ensures consistent evaluation.

⸻

🎯 Problem Statement



Manual assignment checking is:
	•	⏳ Time-consuming
	•	❌ Inconsistent
	•	📉 Hard to track performance trends
	•	🧠 Mentally exhausting for teachers

EduCore solves this by automating the evaluation process using AI and data analytics.

⚙️ System Workflow



1️⃣ Teacher uploads assignment dataset (CSV).
2️⃣ Backend reads and processes data.
3️⃣ AI logic evaluates content quality & structure.
4️⃣ Score and feedback are generated.
5️⃣ Results are displayed on dashboard.

🚀 Installation & Setup Guide

🔹 Step 1: Clone Repository
git clone https://github.com/your-username/EduCore.git

🔹 Step 2: Navigate to Backend Folder
cd backend

🔹 Step 3: Install Dependencies
pip install -r requirements.txt

🔹 Step 4: Run FastAPI Server
uvicorn main:app --reload

🔹 Step 5: Open in Browser
http://127.0.0.1:8000
